### ONE PAGE 반응형 구현

- @media screen, max/min으로 화면의 크기에 따라 레이아웃이 수정되어 보여지도록 구현하였습니다.
- 총 3단계로 구성하였습니다.
- header : 가로형 nav가 사라지고 hamburger menu만 노출하여 정돈하였습니다.
- img : 비디오가 강조될 수 있도록 img는 display: none처리하였습니다.
- text : 비디오에 따라 정돈될 수 있도록 position으로 위치 조정하였습니다.
